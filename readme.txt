== Installation on new project==

1. Repository can be found here https://github.com/redandbluefi/eternia. But you are already reading this so you know where it is
2. Project can be run on any stack as it is just the theme. Local is fast and easy to use.
3. Clone project to the themes folder and remove .git file to empty commit history. Run git init and add new repository $ git remote add origin https://github.com/USER/REPO>.git 
4. Create .env file to the base of the project and copy content from .envexample and add ACF key
5. Run Composer install. Reguires composer version 2 and the presence of .env file with ACF key
6. Run NPM install
7. It should work now :)
